Executive Summary

Heidi Orbit is Adaptive Computing's cutting-edge classroom management platform designed to empower educators with comprehensive real-time monitoring and control capabilities across diverse device ecosystems. Built on modern cloud-native microservices architecture, Heidi Orbit delivers seamless classroom orchestration for Chromebooks, iPads, Windows 11, and MacOS devices in both face-to-face and remote learning environments.

Our solution addresses the critical needs of large-scale educational institutions, providing intuitive teacher controls, robust administrative oversight, and enterprise-grade security—all while maintaining compliance with FERPA and COPPA regulations.

Technical Architecture

Built on modern cloud-native principles, Heidi Orbit leverages microservices architecture to deliver exceptional performance, reliability, and scalability.

Core Technologies

Infrastructure Kubernetes-orchestrated microservices, auto-scaling, multi-region deployment
Communication WebSocket (real-time), REST API, gRPC for service mesh
Data Layer Distributed database, Redis caching, real-time event streaming
Security TLS 1.3 encryption, OAuth 2.0, JWT tokens, AES-256 data encryption
Network Protocols TCP/IP, UDP, WebRTC for screen sharing, HTTP/2 & HTTP/3
Monitoring Distributed tracing, real-time metrics, automated alerting, health checks

Client Components

Chrome OS Native Chrome extension, lightweight agent, browser API integration
Windows 11 Native service, multi-monitor support, application control hooks
MacOS Native agent, system extension, privacy-compliant monitoring
iPad (iOS) Configuration profiles, Apple Classroom connector, supervised mode agent
Teacher Dashboard Progressive Web App (PWA), responsive design, offline capabilities
