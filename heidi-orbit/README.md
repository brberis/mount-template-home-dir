## Printing to PDF

**Important:** These documents must be printed to PDF using **Google Chrome** for optimal formatting.

### Instructions

1. Open the HTML file in **Google Chrome**
2. Press `Cmd + P` (Mac) or `Ctrl + P` (Windows) to open the print dialog
3. Select **"Save as PDF"** as the destination
4. Configure the following settings:
   - **Layout:** Portrait
   - **Paper size:** Letter
   - **Margins:** Default
   - **Background graphics:** âœ“ Enabled (check this box)
5. Click **Save**

### Why Chrome?

- Chrome provides the best CSS print media query support
- Ensures proper page breaks and formatting
- Maintains background colors and gradients
- Consistent rendering of Font Awesome icons
